Простая программа для хранения контактов с замером времени операций.

Старт
Вариант 1 (просто запустить):
Скачать phonebook.exeи запустить

Вариант 2 (собрать самому):
Открой Visual Studio
Создай проект C++
Скинуть туда содержимое папки src


Основные функции
=== МЕНЮ ===
1. Показать все контакты
2. Добавить контакт
3. Удалить контакт
4. Поиск по имени
5. Сортировать по имени
6. Удалить дубликаты
7. Сохранить и выйти

Что делает каждая цифра:
1 Показать контакты
Показывает список всех контактов:

Пример вывода:

Имя: Иван Иванов
Телефон: +7 999 123-45-67
Email: ivan@mail.ru
Теги: друг работа

2 Добавить контакт
По шагам добавляет новый контакт:
Введи имя (нельзя оставить пустым)
Введи телефон (минимум 7 цифр, можно +, -, пробелы)
Введи email (должен содержать @ и точку)
Введи теги через запятую (можно пропустить)

Пример:

Введите имя: Мария Петрова
Введите телефон: +7 916 555-44-33
Введите email: maria@gmail.com
Теги: работа, друг
Контакт добавлен. (15 мс)

3 Удалить контакт
Удаляет контакт по точному имени:
Вводишь имя → программа ищет и удаляет
Если имя найдено — удаляет
Если нет — показывает ошибку

Пример:
text
Имя для удаления: Иван Иванов
Контакт удален. (2 мс)

4 Найти контакты
Ищет контакты по началу имени:
Вводишь начало имени (например: ива)
Находит всех Ивановых, Иванов и т.д.
Регистр не важен (ИВА = ива)
Показывает время поиска

Пример:
text
Поиск по имени: ива
Найдено: 3
Время: 5 мс

5 Сортировать по имени
Сортирует все контакты по алфавиту:
Регистр не учитывается
Показывает время сортировки

Пример:
text
Отсортировано. Время: 10 мс

6 Убрать дубли

Удаляет одинаковые контакты:
Находит контакты с одинаковыми именами
Удаляет дубликаты
Показывает сколько удалил

Пример:
text
Удалено дублей: 2 (Время: 8 мс)

7 Выход
Сохраняет все изменения в файл phonebook.json и закрывает программу.

Файлы
phonebook.json — здесь хранятся все контакты
Формат JSON (можно открыть в блокноте)

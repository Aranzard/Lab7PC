Простая программа для хранения контактов с замером времени операций.  

Старт  
Вариант 1 (просто запустить):  
Скачать phonebook.exeи запустить  

Вариант 2 (собрать самому):  
Открой Visual Studio  
Создай проект C++  
Скинуть туда содержимое папки src  
Примечание: программа не отображает кириллицу  

Сначала вводится имя json файла. Если такого файла нет, то открывается temp.json  
Основные функции  
=== МЕНЮ ===  
1. Показать все контакты  
2. Добавить контакт  
3. Удалить контакт  
4. Поиск по имени  
5. Сортировать по имени   
6. Удалить дубликаты  
7. Сохранить и выйти  

Что делает каждая цифра:  
1 Показать контакты  
Показывает список всех контактов:  

Пример вывода:  
Имя: Ivan Ivanov  
Телефон: 1(897)912-31-39  
Email: ivan@mail.ru  
Теги: friend, work  

2 Добавить контакт  
По шагам добавляет новый контакт:  
Введите имя (нельзя оставить пустым)  
Введите телефон (n(nnn)nnn-nn-nn вводить номер с данной маской. Вместо n любые цифры)  
Введите email (должен содержать один @ и одну точку, а также любые знаки между ними)  
Введите теги через запятую (можно пропустить)  

Пример:  
Введите имя: Maxim Petrov  
Введите телефон: +7 916 555-44-33  
Введите email: maxim@gmail.com  
Теги: работа, друг  
Контакт добавлен. (15 мс)  

3 Удалить контакт  
Удаляет контакт по точному имени:  
Вводишь имя → программа ищет и удаляет  
Если имя найдено — удаляет  
Если нет — показывает ошибку  

Пример:  
text  
Имя для удаления: Ivan Ivanov  
Контакт удален. (2 мс)  

4 Найти контакты  
Ищет контакты по началу имени:  
Вводишь начало имени (например: iva)  
Регистр не важен (IVA = iva)  
Показывает время поиска  

Пример:  
text  
Поиск по имени: iva  
Найдено: 3 (и выводит эти контакты)  
Время: 5 мс  

5 Сортировать по имени  
Сортирует все контакты по алфавиту:  
Регистр не учитывается  
Показывает время сортировки  

Пример:  
text  
Отсортировано. Время: 10 мс  

6 Дедупликация  
Удаляет одинаковые контакты:  
Находит контакты с одинаковыми именами  
Удаляет дубликаты  
Показывает сколько удалил и время, за которое удалил  

Пример:  
text  
Удалено дублей: 2 (Время: 8 мс)  

7 Выход  
Сохраняет все изменения в файл temp.json и закрывает программу.  

Файлы  
temp.json — здесь хранятся все итоговые контакты  
Формат JSON (можно открыть в блокноте)

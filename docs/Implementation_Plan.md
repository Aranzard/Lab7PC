Implementation Plan: Телефонный справочник
1. Краткое описание задачи
Разработать консольное приложение "Телефонный справочник" для управления контактами с поддержкой поиска по префиксу имени, сортировки по алфавиту и дедупликации похожих записей. Приложение должно обрабатывать данные в формате JSON, содержащие информацию о контактах (имя, телефон, email, теги).

2. Цель и ожидаемый результат
Цель: Создать работоспособное консольное приложение с базовыми функциями телефонного справочника и возможностью измерения производительности операций.

Ожидаемый результат:

Консольное приложение на Python с CLI интерфейсом

Модуль для обработки контактов (чтение/запись JSON)

Реализованные алгоритмы поиска и сортировки

Отчет о производительности префиксного поиска

Тестовый набор данных и документация по использованию

Примеры файлов и спецификации:
Входной формат (input.json):

json
[
  {
    "name": "Иванов Иван",
    "phone": "+7 (999) 123-45-67",
    "email": "ivanov@example.com",
    "tags": ["коллега", "друг"]
  },
  {
    "name": "Петров Петр",
    "phone": "8-800-555-35-35",
    "email": "petrov@gmail.com",
    "tags": ["работа"]
  }
]
Спецификация команд:

bash
# Основное использование
python phonebook.py --input contacts.json --find "Ива"

# Дополнительные команды
python phonebook.py --input contacts.json --sort
python phonebook.py --input contacts.json --deduplicate --threshold 0.8
python phonebook.py --input contacts.json --stats
python phonebook.py --input contacts.json --validate

Вариант 16: Телефонный справочник   

Задача:
Разработка консольного приложения для управления телефонным справочником с поддержкой:  
Добавления/удаления/редактирования контактов    
Поиска и сортировки контактов   
Дедупликации похожих записей    
Сохранения и загрузки данных в формате JSON 
Измерения времени выполнения операций   

Язык программирования: C++  
Среды разработки: Visual Studio 2022, VS Code   

Структура json: 
[   
  { 
    "name": "Иван Иванов",                    // string, обязательное   
    "phone": "1(123)456-78-90",               // string, обязательное, строгий формат   
    "email": "ivan@example.com",              // string, обязательное, валидный email   
    "tags": [                                 // array of strings, необязательное   
      "друг",                                 // string, уникальный в пределах контакта 
      "работа"                                // string
    ]   
  },    
  // ... другие контакты    
]   

Правила валидации:  
Имя: не пустое, может содержать пробелы 
Телефон: строго 15 символов, формат 1(123)456-78-90 
Позиции цифр: 0,2,3,4,6,7,8,10,11,13,14 
Позиции символов: '(' на 1, ')' на 5, '-' на 9 и 12     
Email: содержит один '@' и одну '.' после '@', без пробелов   
Теги: уникальные строки, регистрозависимые  

В программе присутствуют функции:   
readJSON считывание контактов из json файла  
writeJSON запись контактов в введенный файл 
findContactsByPrefix поиск контактов по началу имени
sortContactsByName регистронезависимая сортировка контактов по имени
deduplicateContacts удаление одинаковых имён
showContacts вывод всех контактов в консоль
addContact добавление контакта с учетом валидации
deleteContact удаление контакта

Предложения по улучшению:
поддержка кириллицы
уменьшение времени считывания json файла
поиск по номеру, email, тегам

Вывод: 
Разработано консольное приложение для управления телефонным справочником. Реализована система валидации данных с проверкой форматов.
